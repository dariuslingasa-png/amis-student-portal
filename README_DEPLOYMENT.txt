🆘 TROUBLESHOOTING:

Problem: "heroku: command not found"
Solution: Install Heroku CLI and restart terminal

Problem: "App name taken"
Solution: Choose different name when prompted

Problem: "Database error"
Solution: Run: heroku config (check CLEARDB_DATABASE_URL)

Problem: "500 error on live site"
Solution: Run: heroku logs --tail
